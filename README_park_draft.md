# Game Finder

간단한 **HTML/CSS 기반 게임 장르 소개 웹페이지**입니다.  
사용자가 원하는 게임 장르를 선택하면, 해당 장르와 관련된 대표 게임 정보를 확인할 수 있도록 구성했습니다.

## 프로젝트 소개

이 프로젝트는 여러 게임 장르를 한곳에서 살펴볼 수 있도록 만든 정적 웹페이지입니다.  
메인 페이지에서 장르를 선택하고, 세부 장르 페이지를 따라가며 대표 게임 정보를 확인할 수 있습니다.

## 주요 기능

- 메인 페이지에서 게임 장르 선택
- 장르별 / 세부 장르별 페이지 이동
- 대표 게임 목록 확인
- 외부 정보 페이지(예: 위키백과)로 연결
- 장르 분위기에 맞춘 간단한 CSS 스타일 적용

## 페이지 구성

### 메인 페이지
- `Home.html`
- 진입 페이지 역할
- 다음 4가지 장르로 이동 가능
  - Classic
  - Shooting
  - RPG
  - AOS

### 장르 페이지
- `Classic.html` : 고전 게임 소개
- `Shooting.html` : 슈팅 장르 소개
- `RPG.html` : RPG 장르 소개
- `AOS.html` : AOS 장르 소개

### 세부 장르 페이지
- `Bullet.html` : 탄막 슈팅
- `FPS.html` : 1인칭 슈팅
- `TPS.html` : 3인칭 슈팅
- `MMORPG.html` : 대규모 다중 사용자 온라인 RPG
- `Rogue-like.html` : 로그라이크

## 실행 방법

별도의 설치 없이 브라우저에서 바로 실행할 수 있습니다.

### 방법 1. 파일로 실행
1. 레포지토리를 다운로드하거나 클론합니다.
2. `18011736/Home.html` 파일을 브라우저에서 엽니다.

### 방법 2. 로컬 서버로 실행
정적 파일 서버를 사용하면 경로 문제 없이 더 편하게 확인할 수 있습니다.

예시:

```bash
# Python 3 기준
cd 18011736
python -m http.server 8000
```

브라우저에서 `localhost:8000/Home.html` 로 접속하면 됩니다.
