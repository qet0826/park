# Game Finder

A simple static website that helps users browse different game genres and discover example titles through linked pages.

This project is built with HTML and CSS and is organized as a small genre-based game recommendation site. The main page, `Home.html`, introduces **Game Finder** and links to four main categories: **Classic**, **Shooting**, **RPG**, and **AOS**.

## Overview

**Game Finder** is a beginner-friendly web project that presents game genres in a simple navigation structure.
Users can move from the home page to genre pages, then to more specific subgenre pages, and finally view example games with external reference links.

The site appears to have been created on **2019/04/05** by **Park Jung Wook**, based on the text shown on the home page.

## Features

- Static multi-page website built with HTML and CSS
- Genre-based navigation for browsing game types
- Subgenre pages for more detailed exploration
- External links to example games and reference pages
- Lightweight structure that can be opened directly in a browser

## How to Run

Because this is a static website, you can run it very easily:

1. Clone or download the repository.
2. Open the `18011736` folder.
3. Launch `Home.html` in your web browser.

You can also use a local development server such as VS Code Live Server for easier navigation and testing.
