# CSS 스타일시트 안내

이 폴더에는 **Game Finder** 미니 웹 프로젝트에 사용된 스타일시트 파일들이 들어 있습니다.
각 CSS 파일은 사이트의 서로 다른 페이지 구간에 연결되어 있으며, 각 페이지 그룹이 고유한 분위기와 시각적 특징을 가지도록 구성되어 있습니다.

## 개요

`css` 디렉터리에는 총 3개의 파일이 있습니다.

- `H.css` - **홈 페이지** 스타일
- `M.css` - **카테고리 개요 페이지** 스타일
- `G.css` - **장르/상세 페이지** 스타일

## 파일 설명

### `H.css`
메인 시작 페이지인 `Home.html`에 사용됩니다.

주요 특징:
- 첫 화면에 어울리는 깔끔하고 단순한 레이아웃
- 가운데 정렬된 헤더와 내비게이션
- 게임 카테고리 링크마다 서로 다른 hover 색상 적용
- 밝은 톤의 헤더 배경과 강조된 소개 영역 구성

이 파일은 홈 페이지가 프로젝트 전체를 안내하는 간단한 내비게이션 허브처럼 보이도록 만드는 역할을 합니다.

### `M.css`
`Shooting.html`, `RPG.html` 같은 중간 단계의 카테고리 페이지에 사용됩니다.

주요 특징:
- 청록색 계열의 헤더 스타일
- 나무 질감 배경 이미지 사용
- 카테고리 설명에 적합한 테이블 기반 레이아웃
- 뚜렷한 섹션 테두리와 단순한 hover 효과
- `.back` 클래스를 활용한 뒤로가기 버튼 스타일

이 스타일시트는 세부 장르로 이동하기 전에 하위 장르를 소개하는 메뉴형 페이지에 맞춰 설계되었습니다.

### `G.css`
`Classic.html`, `AOS.html`, `Bullet.html`, `FPS.html`, `TPS.html`, `MMORPG.html`, `Rogue-like.html` 같은 상세 장르 페이지에 사용됩니다.

주요 특징:
- 금속 느낌의 패턴 배경 이미지 사용
- 색 테두리가 적용된 흰색 콘텐츠 박스
- 외부 게임 링크에 대한 공통 기본 링크 스타일
- 버튼 및 링크 변형을 위한 `.R`, `.M`, `.TH`, `.TM` 클래스 포함
- 이미지 중심 콘텐츠 페이지에 맞춘 넉넉한 내비게이션 간격

이 파일은 특정 게임 목록과 참고 링크를 보여주는 상세 페이지들의 공통 시각 스타일을 담당합니다.

## 폴더 구조

```text
css/
├── G.css
├── H.css
└── M.css
```

## 참고 사항

- 이 프로젝트는 **순수 HTML과 CSS만 사용**하여 제작되었습니다.
- 스타일은 하나의 전역 스타일시트로 합치지 않고, 페이지 역할에 따라 분리되어 있습니다.
- 스타일시트에서 참조하는 배경 이미지는 상위 폴더와 나란한 `img` 폴더에 저장되어 있습니다.

## 개선 아이디어

추후 유지보수를 고려한다면 다음과 같이 구조를 개선할 수 있습니다.

- 공통 레이아웃 규칙을 담은 기본 스타일시트 1개
- 페이지별 테마를 담당하는 스타일시트 분리
- `home.css`, `category.css`, `detail.css`처럼 더 직관적인 파일명 사용

이렇게 정리하면 처음 프로젝트를 보는 사람도 구조를 더 쉽게 이해할 수 있습니다.
